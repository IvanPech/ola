Primera página WEB

Por el inicio

Como mi primera página, solo escribiré poco texto para centrarme más en el etiquedado de HTML y comprobar mis aprendizajes en el desarrollo web.

Las etiquetas báscias que usaré

    La etiqueta html encierra todo el contenido que debe de llevar una página web

    La etiqueta head es la cabezera que tiene los metadatos que definen como se va a procesar y visualizar la página web.

    La eqtiqueta title es la etiqueta que usamos para escribir el título de la página web.

    La etiqueta body es el cuerpo de la página.

    Las etiquetas h1, h2, h3... son los titulos que se usan para hacer titulos, siendo el h1 el de más valor o más grande.

    La etiqueta strong es para remarcar con negritas lo que escribamos dentro de la etiqueta.

    La etiqueta em es para enfatizar lo que escribamos dentro de la etiqueda poniendolo en cursiva

    Estas son las etiquetas más básicas que se utilizan para crear o desarrollar una página web y que estoy usando para darle lugar a esta mi primera página web
